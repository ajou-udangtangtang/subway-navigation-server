# 01. 프로젝트 개요

## 1.1 프로젝트 소개

### 1.1.1 과제명

**지하철역 보행지원 스마트폰 어플리케이션**

### 1.1.2 한 줄 요약

지하철역 내부에서 시각장애인이 자신의 위치와 방향을 정확히 인지하지 못하는 문제를 해결하기 위해, **Wi-Fi Fingerprinting 기반 위치 추정**과 **노드 그래프 기반 경로 안내**를 결합하여 진동·음성 피드백으로 길을 안내하는 안드로이드 어플리케이션 시스템.

### 1.1.3 시스템 한 줄 정의

> "지하철역 안 시각장애인용 내비게이션"

차량용 내비게이션과 동일한 구성 원리를 따르되, 실외 GPS 대신 실내 Wi-Fi 신호로 위치를 추정하고, 화면 대신 진동·음성으로 안내한다.

| 구성 요소 | 차량 내비게이션 | 본 시스템 |
|---|---|---|
| 위치 인식 | GPS | Wi-Fi Fingerprinting + KNN |
| 경로 탐색 | 도로 지도 | 노드 그래프 + Dijkstra |
| 안내 매체 | 화면, 음성 | **진동, 음성 (TalkBack 포함)** |

## 1.2 추진 배경

### 1.2.1 시각장애인의 이동권 문제

시각장애인은 실내 공간, 특히 **지하철역과 같이 구조가 복잡한 지하 공간에서 자신의 위치와 진행 방향을 정확히 인지하기 어려운** 환경에 노출되어 있다. 점자블록은 정해진 경로를 따라 안내하는 데에는 유효하지만, 갈림길의 방향 결정이나 목적지까지의 경로 안내에는 한계가 있다.

### 1.2.2 실내 환경에서의 GPS 한계

일반적으로 실외 보행자 안내는 **GPS** 기반으로 수행되지만, 다음과 같은 이유로 지하철역 내부에서는 활용이 어렵다.

- **신호 차단**: 지하 공간에서는 GPS 위성 신호가 거의 수신되지 않는다.
- **정밀도 부족**: 수신이 가능한 1층 진입부에서도 오차가 3~10m 수준으로, 좁은 통로·갈림길 단위 안내에는 부적합하다.

### 1.2.3 대체 위치 추정 기법의 필요성

GPS 한계를 보완하기 위한 실내 위치 추정 기법으로는 **Wi-Fi Fingerprinting**, **BLE 비콘**, **지자기 매핑** 등이 있다. 본 프로젝트에서는 **별도 인프라 구축이 불필요하고 기존 지하철역의 Wi-Fi AP를 그대로 활용 가능한 Wi-Fi Fingerprinting** 방식을 채택한다.

### 1.2.4 시각장애인 친화 인터페이스의 필요성

화면을 인지할 수 없는 사용자를 대상으로 하므로, 모든 안내는 **진동(Haptic)** 과 **음성(TTS)** 으로 전달되어야 한다. 또한 안드로이드의 **TalkBack** 보조 기능과 호환되도록 UI를 설계하여, 사용자가 평소 사용하던 스크린 리더 환경에서 그대로 사용 가능해야 한다.

## 1.3 프로젝트 목적

> 시각장애인이 자신의 위치와 방향을 정확히 인지할 수 없어 올바른 경로로 이동할 수 있도록 지원하는 것을 목적으로 한다. *(원본 요구사항 명세서 中)*

본 프로젝트의 구체적 목적은 다음과 같다.

1. **위치 인지 지원**: 지하철역 내부의 주요 지점(개찰구, 환승 통로, 출구 등)을 사용자가 인식할 수 있도록 한다.
2. **방향 안내 지원**: 사용자가 다음 목적 지점으로 향할 때, 진동의 강·약을 통해 방향을 직관적으로 전달한다.
3. **위험 회피 지원**: 에스컬레이터 등 위험 구간을 회피한 안전 경로를 자동으로 산출한다.
4. **자율 이동 보조**: 동행자의 도움 없이도 사용자가 자율적으로 지하철역을 이용할 수 있도록 한다.

## 1.4 기대 효과

### 1.4.1 사용자 측면

- **자율적 이동 능력 향상**: 동반자나 역무원 도움 없이도 지하철역 내부 이동이 가능해진다.
- **안전성 향상**: 위험 구간(에스컬레이터 등) 회피 및 사전 안내를 통해 사고 위험을 감소시킨다.
- **정신적 부담 완화**: "현재 어디인지 모르는 상태"가 줄어듦으로써 이동 중 불안감을 완화한다.

### 1.4.2 사회적 측면

- **이동권 보장**: 시각장애인의 대중교통 접근성을 높여 이동권 보장에 기여한다.
- **사회 참여 확대**: 자율적인 이동 수단 확보는 교육·취업 등 사회 활동 참여 기회 확대로 이어진다.

### 1.4.3 기술적 측면

- **Wi-Fi Fingerprinting의 실증 적용**: 별도 인프라 추가 없이 기존 지하철 Wi-Fi AP를 활용한 실내 위치 추정의 실현 가능성을 검증한다.
- **재사용 가능한 안내 모델**: 노드 그래프 기반 경로 안내 구조는 다른 실내 공간(병원, 대형 쇼핑몰, 공항 등)에도 응용 가능하다.

## 1.5 본 보고서의 범위

본 보고서는 4명으로 구성된 졸업프로젝트 팀의 작업물 중 **서버 파트(Flask 기반 백엔드)** 의 설계 및 구현 계획을 다룬다. 안드로이드 앱(클라이언트) 파트는 본 보고서의 직접적인 작성 범위에 포함되지 않으나, 인터페이스 정의에 필요한 범위 내에서 함께 기술한다.
